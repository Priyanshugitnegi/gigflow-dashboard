# GigFlow Dashboard

GigFlow is a full-stack Lead Management Dashboard built using React and Django REST Framework.

## Live Demo

Frontend:
https://gigflow-dashboard-nu.vercel.app

Backend:
https://web-production-c139e.up.railway.app

---

## Features

- JWT Authentication
- User Registration
- User Login
- Create Leads
- View Leads
- Update Lead Status
- Delete Leads
- Search Leads
- Responsive Dashboard UI

---

## Frontend Tech Stack

- React.js
- Axios
- React Router
- CSS

---

## Backend Tech Stack

- Django REST Framework
- JWT Authentication
- SQLite
- Railway Deployment

---

## Deployment

- Frontend → Vercel
- Backend → Railway

---

## Local Setup

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
